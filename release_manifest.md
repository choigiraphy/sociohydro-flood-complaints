## Excluded from public release

- Raw complaint texts
- Complaint-level restricted spreadsheets
- Manuscript draft history
- Any files governed by the ACRC memorandum of understanding

## Pre-release checklist

1. Create the public GitHub repository.
2. Push the release package.
3. Create a Zenodo release from the GitHub repository.
4. Record the GitHub URL and Zenodo DOI.
5. Replace temporary manuscript wording in the Open Research section with the final URLs and DOI.
